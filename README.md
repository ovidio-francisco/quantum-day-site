# UNISO Quantum Day 2026 — site estático

Estrutura inicial em **HTML + CSS + JavaScript**, com:

- layout responsivo
- contador regressivo
- animações com **AOS** via CDN
- fundo animado em `canvas`
- placeholders para patrocinadores
- seções prontas para programação, inscrição, local e galeria futura

## Arquivos

- `index.html` — estrutura principal do site
- `style.css` — estilos globais
- `script.js` — contador, menu mobile e partículas
- `assets/img/logo.png` — logo fornecido
- `assets/img/logo.jpeg` — logo fornecido
- `assets/img/placeholder-sponsor-*.svg` — logos temporários

## O que você deve trocar depois

1. Link de inscrição em `index.html`
2. Logos placeholder por logos reais dos patrocinadores
3. Textos finais de palestrantes, minicursos e parceiros
4. Fotos da galeria após o evento

## Bibliotecas usadas

- **Google Fonts** — Orbitron + Inter
- **AOS (Animate On Scroll)** — gratuita via CDN

## Observação

O conteúdo textual foi elaborado a partir do objetivo do site, do público-alvo informado e do midia kit do evento.
